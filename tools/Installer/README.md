# Flashable_UEFI_Installer

edk2-msm UEFI A/B Installer Zip Template

Forked and modified from [twrp_abtemplate by osm0sis](https://github.com/osm0sis/twrp_abtemplate).

## Packing How-to:
Put a prebuilt UEFI fd file with BootShim and Padding from [edk2-msm](https://github.com/edk2-porting/edk2-msm) named *_UEFI-Padding.fd under this folder and pack as a Zip file.

## Installation:
Flash the packed Zip file using TWRP or Magisk Manager.
